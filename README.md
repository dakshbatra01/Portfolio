# Daksh Batra Portfolio

This is a personal developer portfolio web application structured like a mock Operating System interface (DakshOS). It showcases projects, education, achievements, skills, and contains an interactive terminal console with a custom command line interface.

## Tech Stack
- Frontend: React.js, Vite
- Styling: Custom Vanilla CSS
- Email Service: EmailJS
- Packages: Lucide React

## Features
- Interactive mock desktop interface
- Terminal CLI supporting custom commands (help, ls, cat, whoami, neofetch, contact, ask, social, open)
- File Manager simulation
- Web Browser app displaying real public projects
- Retro game integrations (Snake, Tic-Tac-Toe)
- Custom boot sequence screen

## Project Structure
- src/components/apps: Standalone application widgets (Browser, Terminal, FileManager, games)
- src/components/widgets: Desktop UI elements (ContextMenu, Notifications, window management)
- src/data: Core data configurations (resume details, file system layout)
